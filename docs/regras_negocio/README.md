# Regras de Negócio

## Objetivo

Documentar todas as regras operacionais utilizadas pelo Supervisor AI.

Cada processo operacional deverá possuir um documento independente.

---

# Estrutura dos documentos

Cada documento deverá conter, sempre que aplicável:

- objetivo;
- conceitos;
- regras;
- exceções;
- fluxo;
- dados necessários;
- observações.

---

# Processos

- Upgrade
- RV
- Extras
- Qualidade
- Reincidência
- CSAT
- Cancelamentos
- Escalas

---

# Princípio

O Supervisor AI implementará apenas regras documentadas.

Sempre que houver dúvidas sobre um processo operacional, ele deverá ser revisado antes da implementação.
