# PRD

## Objetivo

Documentar a visão funcional do Supervisor AI.

O PRD descreve o problema que o produto resolve e quais funcionalidades deverão existir.

---

# Conteúdo

- visão do produto;
- objetivos;
- personas;
- casos de uso;
- jornadas;
- requisitos funcionais.

---

# Importante

O PRD não define implementação técnica.

Ele descreve apenas o comportamento esperado do produto.
