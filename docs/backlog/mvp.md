# MVP - Supervisor AI

## Objetivo

Entregar uma primeira versão funcional do Supervisor AI capaz de reduzir significativamente o trabalho manual realizado atualmente pelos supervisores.

O MVP não pretende substituir completamente todos os sistemas utilizados pela empresa.

Seu foco é automatizar processos repetitivos e centralizar informações utilizadas diariamente.

---

# Critérios para o MVP

O MVP será considerado concluído quando for capaz de:

- Coletar automaticamente os dados necessários;
- Processar as regras de negócio existentes;
- Atualizar indicadores automaticamente;
- Eliminar a atualização manual das planilhas principais;
- Disponibilizar dashboards para acompanhamento.
