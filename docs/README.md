# Documentação do Supervisor AI

## Objetivo

A pasta `docs` concentra toda a documentação necessária para o desenvolvimento, manutenção e evolução do Supervisor AI.

O objetivo é garantir que o conhecimento do projeto esteja documentado antes da implementação, permitindo que qualquer desenvolvedor ou IA compreenda rapidamente o funcionamento do sistema.

# Princípios

A documentação deve ser considerada a principal fonte de conhecimento do projeto.

Sempre que possível:

- documentar antes de implementar;
- revisar antes de programar;
- registrar decisões importantes em ADRs.

---

# Filosofia do Projeto

O Supervisor AI é um software orientado a processos.

Antes de escrever código, busca-se compreender completamente o funcionamento da operação e documentar esse conhecimento de forma estruturada.
