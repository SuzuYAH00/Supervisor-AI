# Pesquisas

## Objetivo

Armazenar estudos e informações levantadas durante o desenvolvimento do Supervisor AI.

---

# Exemplos

- observações operacionais;
- estudos sobre sistemas;
- pesquisas realizadas;
- comportamento dos usuários;
- ideias técnicas.

---

# Observação

As pesquisas representam conhecimento em construção.

Após validadas, poderão originar:

- regras de negócio;
- backlog;
- ADRs;
- arquitetura.
