# Reuniões

## Objetivo

Registrar decisões tomadas durante reuniões relacionadas ao Supervisor AI.

---

# Conteúdo

Cada reunião deverá registrar:

- data;
- participantes;
- decisões;
- pendências;
- próximos passos.

---

# Objetivo

Manter histórico das decisões que influenciaram a evolução do projeto.
