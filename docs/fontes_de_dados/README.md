# Fontes de Dados

## Objetivo

Documentar todas as origens de dados utilizadas pelo Supervisor AI.

---

# O que será documentado

Para cada sistema deverão ser registrados:

- origem;
- tipo de integração;
- frequência de atualização;
- informações disponíveis;
- limitações;
- observações.

---

# Fontes previstas

- MK Solutions
- Google Sheets
- Looker Studio
- Witto
- WhatsApp

Outras integrações poderão ser adicionadas futuramente.

---

# Princípio

O Supervisor AI deve utilizar os sistemas oficiais como fonte de verdade sempre que possível.
